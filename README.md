# Food Ordering App

## Overview
This is a feature-rich food ordering application that allows users to browse menus, place orders, and track their deliveries seamlessly. The user panel is fully implemented, providing an intuitive and smooth experience.

## Features
- **User Panel**
  - User authentication (login/SugnUp)
  - Browse restaurant menus with images and descriptions
  - Search food items
  - Add items to cart and modify orders
  - Place orders and track order status
  - Profile management
  
- **Upcoming Features**
  - Admin panel for managing orders and menus
  - Payment gateway integration
  - Live order tracking
  - Push notifications

## Technologies Used
- **Frontend:** Android (Kotlin)
- **UI Components:** XML Layouts , Figma

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/food-ordering-app.git
   ```
2. Open the project in Android Studio.
3. Build and run the app on an emulator or physical device.
